# 🌐 notPlayzz Website
The official home for notPlayzz texture packs, hosted on GitHub Pages.

## 🔗 Live Site
[View the notPlayzz Website](https://isiguzoflorence521-gif.github.io/texture-packs/)

## 🚀 About the Project
This website is a gallery and download hub for high-performance Bloxd.io texture packs. 
- **Updates**: 10+ new textures added every single day!
- **Tech Stack**: Built with HTML5, CSS3, and JavaScript.

## 📸 Preview
![Website Screenshot](./assets/site-preview.png)

## 📂 Project Structure
- `index.html` - The main landing page.
- `/assets` - Images and icons used on the website.
- `/packs` - Storage for the actual texture pack folders (e.g., `notPlayzz-9K-8x`).

## 🛠 Local Development
To run this site locally:
1. Clone the repo: `git clone https://github.com`
2. Open `index.html` in your browser.
